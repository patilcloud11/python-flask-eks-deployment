# Python Flask App Deployment on AWS EKS

## Overview
Containerized Python Flask application deployed on AWS EKS using Docker and Amazon ECR.

## Tech Stack
- Python Flask
- Docker
- Amazon ECR
- Amazon EKS
- Kubernetes

## Features
- Containerized application
- Kubernetes Deployment
- NodePort Service exposure
- Private image registry integration

## Access
Application exposed via worker node public IP and NodePort.
